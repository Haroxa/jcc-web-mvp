# 项目状态

最后更新：2026-06-19 10:05

## 项目类型和技术栈

- 类型：可部署 Web 应用。
- 前端：React + TypeScript + Vite。
- 数据层：Drizzle ORM。
- 本地数据库：SQLite。
- 云端目标：Cloudflare Workers + D1 + R2。
- 备选部署：Docker 云服务器。
- UI 方向：朴素后台风格，优先功能完整。

## 当前目标

先做一个可靠的 MVP 骨架，跑通真实直播核心链路：

```text
创建场次
  -> 定榜
  -> 生成/确认名单
  -> 对局锁牌
  -> 存票/取票/现刷
  -> 结算
  -> 游客查看公开存票榜
```
