# 风险清单

最后更新：2026-06-20 12:01

## 风险等级

- `P0`：可能造成重要数据丢失、公开泄露、核心流程不可用。
- `P1`：影响核心功能正确性或真实直播使用。
- `P2`：影响体验、维护性或局部功能。
- `P3`：低风险优化项。

## 风险概览

| 类别 | 当前判断 | 需要关注的触发点 |
| --- | --- | --- |
| 数据正确性 | 最高优先级 | 票务流水、结算、作废、回退、锁牌释放 |
| 权限和公开范围 | 最高优先级 | 游客页、公开榜、公开截图、管理员代操作 |
| 部署和环境 | Pages Functions 和远程 D1 已验证，R2 暂缓 | Cloudflare Pages Functions、D1、Docker 备选 |
| 使用效率 | 已成为当前关键风险 | 手机端操作、直播中快速录入、当前场次上下文、页面跳转过多 |
| 维护追溯 | 已有 Git 和文档规则，需持续执行 | 每次变更是否同步日志、问题、风险、计划 |
