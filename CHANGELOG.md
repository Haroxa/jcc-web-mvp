## 常用 Git 命令

### 查看当前文件状态

```powershell
git status
```

用途：查看哪些文件新增、修改或还没有提交。

### 把文件加入本次提交

```powershell
git add .
```

用途：把当前文件夹里的改动加入待提交列表。

### 提交一个版本

```powershell
git commit -m "初始化项目版本管理"
```

用途：保存一个可以回看的版本点。

### 查看提交历史

```powershell
git log --oneline
```

用途：用简短形式查看之前提交过哪些版本。

### 查看某个文件改了什么

```powershell
git diff
```

用途：在提交前检查具体修改内容。

### 查看最近一次提交详情

```powershell
git show --stat
```

用途：查看最近一次提交改动了哪些文件。
