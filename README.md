# 水友赛记录助手 MVP

这是一个可直接部署为静态网页的最小可行版本，用来验证“截图管理 + 锁牌管理 + 存票管理”是否真的适合水友赛使用。

## 当前范围

- 截图管理：上传截图、填写玩家/标签/备注、搜索、删除。
- 锁牌管理：内置 8 个玩家和 8 张五费占位卡，支持锁牌、释放、淘汰后自动释放。
- 存票管理：记录存票/取票、备注、自动余额排行。
- 数据保存：使用浏览器本地存储。
- 数据迁移：支持 JSON 导入和导出。

## 使用方式

直接打开 `index.html` 即可使用，不需要安装依赖，也不需要启动服务。

## 部署建议

当前版本是纯静态文件，可以放到 Cloudflare Pages、Vercel 或 GitHub Pages。正式发布前建议先用 Cloudflare Pages 做国内访问测试。

## 明确不包含

- 不包含登录注册。
- 不包含云端数据库。
- 不包含多人实时协作。
- 不包含官方素材、英雄图标、装备图标。
- 不包含截图自动识别。

## 注意

截图以本地浏览器数据形式保存，图片太多或太大时可能触发浏览器存储限制。重要数据请定期使用“导出数据”备份。
