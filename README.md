# 水友赛记录助手 MVP

这是一个面向金铲铲水友赛的本地记录网页，用来验证“截图管理、锁牌管理、存票管理”是否能提升自定义比赛的记录效率。

项目当前定位是轻量 MVP：不需要安装依赖，不需要后端，不需要登录，数据保存在当前浏览器里。

线上测试地址：

```text
https://jcc-web-mvp.pages.dev/
```

## 项目用途

- 截图管理：保存精彩对局截图，记录玩家、标签和备注。
- 锁牌管理：适合 8 人自定义追五费，记录玩家锁定的五费卡，并配置本场玩家和本场五费。
- 存票管理：记录存票、取票和备注，并按当前余额排序。
- 基础资料：集中维护候选玩家、五费名称、五费分类和标签，支持批量文本解析、编号展示、筛选和固定高度分页。
- 数据迁移：支持导出和导入 JSON 数据。

## 技术栈

- HTML
- CSS
- 原生 JavaScript
- `localStorage`

当前没有使用 Vue、React、Vite、npm、后端服务或数据库。

## 启动方式

直接用浏览器打开：

```text
index.html
```

也可以在文件管理器中双击 `index.html`。

## 检查方式

当前项目没有自动化测试脚本。修改后建议手动检查：

- 页面能正常打开。
- 四个页签可以切换。
- 截图记录可以新增、搜索、删除。
- 锁牌记录可以选择、释放、重置。
- 本场玩家和本场五费在锁牌管理页配置，基础资料页只展示当前状态。
- 淘汰玩家的锁牌记录会保留并可修改，但不占用存活玩家的五费选择。
- 锁牌管理会根据淘汰顺序自动显示排名。
- 基础资料可以批量解析玩家和五费文本，并支持筛选、页码分页、固定高度列表和顺序编号。
- 玩家可以在锁牌管理页设置本场参与，锁牌和存票只使用本场玩家。
- 玩家资料支持昵称、抖音名、微信名和游戏名，锁牌页可切换展示方式。
- 五费可以在锁牌管理页设置本场可用，分类和标签可以保存，并同步到锁牌选择。
- 五费资料支持外号，锁牌页可切换正式名或外号优先。
- 锁牌管理支持按五费分类筛选添加选项，五费分类用颜色区分。
- 存票记录可以新增并正确排序。
- 导出和导入 JSON 可用。

## 主要文件说明

- `index.html`：页面结构和三个工具入口。
- `styles.css`：页面布局、颜色、按钮、卡片和响应式样式。
- `app.js`：本地数据、截图、锁牌、存票、导入导出逻辑。
- `AGENTS.md`：Codex 在本项目中的工作规则。
- `CHANGELOG.md`：常用 Git 命令和变更记录。
- `REVIEW.md`：前期执行复盘。
- `data/玩家.txt`：候选玩家原始文本资料。
- `data/五费卡.txt`：五费卡原始文本资料。
- `docs/项目状态.md`：当前功能状态和未验证项。
- `docs/风险清单.md`：当前已知风险和排查建议。

## 部署建议

当前版本是纯静态文件，可以部署到 Cloudflare Pages、Vercel 或 GitHub Pages。

当前已部署到 Cloudflare Pages：`https://jcc-web-mvp.pages.dev/`。

如果主要用户在中国大陆，建议继续收集不同网络下的访问速度和稳定性反馈。正式公开前，请避免使用官方游戏素材，并保留“非官方项目”说明。

## 当前不包含

- 登录注册
- 云端数据库
- 多人实时协作
- 截图自动识别
- 官方素材库
- 支付或会员功能

## 注意事项

截图以浏览器本地数据形式保存，图片太多或太大时可能触发浏览器存储限制。重要数据请定期使用“导出数据”备份。
