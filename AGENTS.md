# AGENTS.md

本文件是 Codex 在本项目中的工作规则。开始修改前请先阅读本文。

## 项目概况

- 项目名称：水友赛记录助手 MVP
- 项目目标：为金铲铲水友赛提供一个轻量网页工具，用来本地记录截图、锁牌和存票信息，先验证真实使用价值。
- 项目类型：纯静态网页 MVP
- 主要技术栈：HTML、CSS、原生 JavaScript、浏览器本地存储
- 当前阶段：本地可用、可静态部署验证

## 常用命令

- 启动方式：直接用浏览器打开 `index.html`
- 测试方式：手动检查截图管理、锁牌管理、存票管理、导入导出
- 构建命令：暂无，不需要构建
- 查看 Git 状态：`git status --short --branch`

## 工作规则

- 修改前先确认相关文件和影响范围。
- 保持改动范围小，不做无关重构。
- 优先沿用当前无依赖、纯静态网页结构。
- 不要默认引入框架、后端、数据库、登录或云同步。
- 修改本地数据结构前，先说明是否影响旧数据。
- 涉及公开部署时，不使用官方游戏素材、Logo、英雄图标或装备图标。
- 修改后运行相关检查；无法运行自动化检查时，说明原因并给出手动检查建议。

## 不要默认修改

- `.git` 目录
- 浏览器中的真实本地数据
- 与当前任务无关的功能文件
- 官方素材、密钥、账号、token
- 部署平台配置，除非用户明确要求

4. 是否还有风险或下一步建议。
